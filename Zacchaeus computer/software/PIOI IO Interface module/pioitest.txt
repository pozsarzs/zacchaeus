PIOITest program
~~~~~~~~~~~~~~~~

  version	v0.1
  licence	EUPL v1.2
  architecture	Z80
  OS		CP/M-80
  compiler	SLR Systems Z80ASM v1.32 (1986)
  author	Pozsar Zsolt <pozsarzs@gmail.com> (2025)

A simple utility to test the PIOI module on the Zacchaeus Microcomputer.
Provides basic functionality for setting addresses, reading inputs, and
controlling outputs.
